Analysis Cohort

master comes from the shared pipeline. Throughout this analysis:

  • Exposure: etoh_hepatitis - alcoholic hepatitis (AH) vs. non-AH
  • Primary outcome (event of interest, cause = 1): death within 90 days
  • Competing risk (cause = 2): liver transplant within 90 days
  • Censoring (cause = 0): alive and not transplanted at 90 days / lost to follow-up

Time and status come from the shared pipeline variables time_to_death_90days_cmp / time_to_death_status_90days_cmp (0 = censored, 1 = death, 2 = transplant).

Show cohort filter 
# Cause coding for the competing-risk event factor `event_cmp`:
#   level 1 "Censored"   -> censoring        (numeric code 0)
#   level 2 "Death"      -> OUTCOME OF INTEREST (numeric code 1)
#   level 3 "Transplant" -> COMPETING RISK   (numeric code 2)
# This ordering matters: tidycmprsk::crr() treats the first level as the
# censoring code, and `failcode` selects which remaining level is the cause
# of interest (all other non-censoring levels become competing risks).
analysis <- master %>%
  filter(!is.na(etoh_hepatitis)) %>%
  mutate(
    etoh_hepatitis = factor(etoh_hepatitis, levels = c(0, 1), labels = c("Non-AH", "AH")),
    event_cmp      = factor(time_to_death_status_90days_cmp,
                            levels = c(0, 1, 2),
                            labels = c("Censored", "Death", "Transplant"))
  )

# Aalen-Johansen non-parametric CIF (used for both the plot and the day-90 estimates below)
ci_unadj <- cuminc(
  Surv(time_to_death_90days_cmp, event_cmp) ~ etoh_hepatitis,
  data = analysis
)

ci_unadj %>%
  ggcuminc(outcome = "Death") +
  labs(
    title    = "Unadjusted 90-day cumulative incidence of death",
    subtitle = "AH vs. non-AH",
    x        = "Days from admission",
    y        = "Cumulative incidence  -  Death",
    caption  = "Outcome: death (cause 1)  |  Competing risk: liver transplant (cause 2)"
  ) +
  add_confidence_interval() +
  add_risktable() +
  scale_x_continuous(breaks = seq(0, 90, by = 30),
                     limits = c(0, 90),
                     expand = c(0, 0)) +
  theme(plot.caption = element_text(face = "italic"))

90-day cumulative incidence of death (AH vs. non-AH)

Non-parametric Aalen-Johansen estimator at t = 90 days, accounting for liver transplant as a competing risk. 95% CIs use the log-log transformation.

ci90_raw <- ci_unadj %>%
  tidy(times = 90) %>%
  filter(outcome == "Death")
